## Instrucciones para el equipo

Para comenzar a trabajar en el proyecto, sigue estos pasos:

1. Clona este repositorio en tu máquina local. Asegúrate de reemplazar `NOMBRE_DEL_EQUIPO` con el nombre de tu equipo:

```bash
git clone https://github.com/DCConehaton/casero-NOMBRE_DEL_EQUIPO
```

2. Navega al directorio del proyecto:

```bash
cd casero-NOMBRE_DEL_EQUIPO
```

3. Abre el proyecto en tu editor de código favorito (por ejemplo, Visual Studio Code):

```bash
code .
``` 

## Para crear tu app con NextJS, AstroJS, etc.

Si quieres crear tu pagina con NextJS, AstroJS u otro framework, puedes crearlo con:

```bash
pnpm create-next-app@latest app 
```

o

```bash
pnpm create astro@latest app
```

> [!CAUTION]
> Si intentas crear el proyecto en la raíz del repositorio, no podrás ya que el repositorio ya tiene archivos. Asegúrate de crear el proyecto dentro de la carpeta `app` para evitar problemas.
>
> Es importante que la aplicación no sea creada con un repositorio git, ya que esto puede causar conflictos con el repositorio principal. Asegúrate de no inicializar un nuevo repositorio git dentro de la carpeta `app`.
